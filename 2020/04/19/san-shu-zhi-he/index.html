三数之和

发表于 2020-04-19 更新于 2020-04-25 分类于 算法与数学

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

 

示例：

给定数组 nums = [-1, 0, 1, 2, -1, -4]，

满足要求的三元组集合为：
[
  [-1, 0, 1],
  [-1, -1, 2]
]

    • 先排序，然后定点，双指针搜索。另外去重。 
    class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        List<List<Integer>> rstList = new ArrayList<List<Integer>>();

        for (int i = 0; i < nums.length - 2; i++) {
            if (nums[i] > 0) {
                break;
            }
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }

            int first = i + 1;
            int end = nums.length - 1;
            while (first < end) {
                int temp = nums[i] + nums[first] + nums[end];
                if (temp > 0) {
                    end--;
                } else if (temp < 0) {
                    first++;
                } else {
                    List<Integer> list = new ArrayList<>();
                    list.add(nums[i]);
                    list.add(nums[first]);
                    list.add(nums[end]);
                    rstList.add(list);
                    first++;
                    end--;
                    while (first < end && nums[first] == nums[first - 1]) {
                        first++;
                    }
                    while (first < end && nums[end] == nums[end + 1]) {
                        end--;
                    }
                }
            }
        }
        return rstList;
    }
}
